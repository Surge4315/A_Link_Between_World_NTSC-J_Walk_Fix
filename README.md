# A_Link_Between_World_NTSC-J_Walk_Fix
Romhack to fix Link's movement in japanese ALBW since romhacking.net doesn't host hacks for japanese games, based on i_hate_drm's [Control Fix](https://www.romhacking.net/hacks/8658/) for PAL and NTSC-U versions. 

![test](https://www.romhacking.net/hacks/3ds/images/8658screenshot1.png)

Patch can be applied using [LunarIPS](https://www.romhacking.net/utilities/240/)
